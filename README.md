# 📍 Recife: Pontos Turísticos

Este projeto é uma plataforma informativa dedicada a explorar as riquezas históricas e culturais da cidade do **Recife, Pernambuco**. Com foco no Recife Antigo, o site oferece uma imersão nos principais monumentos que definem a identidade da "Veneza Brasileira".

---

## 🏛️ Destaques do Roteiro

O site está estruturado para navegar entre os pontos mais emblemáticos da capital pernambucana:

* **📍 Marco Zero:** O coração geográfico da cidade, apresentando a famosa rosa dos ventos de Cícero Dias.
* **🏢 Palácio do Comércio:** Um mergulho na arquitetura e na história econômica da região.
* **🗿 Parque das Esculturas:** Informações sobre as obras de Francisco Brennand localizadas no molhe do porto.

---

## ✨ Funcionalidades

* **Navegação Integrada:** Menu de acesso rápido para diferentes páginas de pontos turísticos.
* **Conteúdo Histórico:** Textos detalhados sobre a revitalização do Recife Antigo, importância geográfica e eventos culturais (como o Carnaval).
* **Design Responsivo:** Estrutura preparada para leitura em diferentes dispositivos através do `main.css`.
* **Acessibilidade Visual:** Uso da tag `<figure>` e atributos `alt` para descrição de imagens.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5**: Estruturação semântica para melhor ranqueamento e leitura.
* **CSS3**: Estilização personalizada mantida em um diretório de ativos (`assets/styles`).
* **LinkedIn Integration:** Link direto no rodapé para contato profissional com a desenvolvedora.

---

## 📂 Estrutura de Arquivos

```text
├── assets/
│   ├── imgs/                  # Imagens dos pontos turísticos (ex: marco-zero.png)
│   └── styles/                # Arquivo main.css para identidade visual
├── index.html                 # Página principal (Marco Zero)
├── palacio-do-comercio.html   # Informações sobre o Palácio
└── parque das esculturas.html # Informações sobre o Parque das Esculturas
```

---

## 🚀 Como Visualizar

1.  Clone o repositório.
2.  Certifique-se de que o caminho das imagens em `assets/imgs/` esteja correto.
3.  Abra o `index.html` em seu navegador para iniciar o tour virtual.
